# 845. 数组中的最长山脉
我们把数组 A 中符合下列属性的任意连续子数组 B 称为 “山脉”：
```shell
B.length >= 3
存在 0 < i < B.length - 1 使得 B[0] < B[1] < ... B[i-1] < B[i] > B[i+1] > ... > B[B.length - 1]
注意：B 可以是 A 的任意子数组，包括整个数组 A。
```
给出一个整数数组 A，返回最长 “山脉” 的长度。如果不含有 “山脉” 则返回 0。

```shell
示例 1：
输入：[2,1,4,7,3,2,5]
输出：5
解释：最长的 “山脉” 是 [1,4,7,3,2]，长度为 5。

示例 2：
输入：[2,2,2]
输出：0
解释：不含 “山脉”。
```

### python双指针模拟上山再下山
```python
# 执行用时：72 ms, 在所有 Python3 提交中击败了93.61% 的用户

class Solution:
    def longestMountain(self, A: List[int]) -> int:
        result = 0
        left = 0
        right = 0

        # 找第一个上山点
        for i in range(1, len(A)):
            if A[i] > A[i - 1]:
                left = i - 1
                right = i
                break

        if right == 0:
            return 0

        # 记录是上山状态还是下山状态
        is_up = True

        while right < len(A):
            if is_up:
                if A[right] > A[right - 1]:
                    right += 1
                elif A[right] == A[right - 1]:
                    # 迭代寻找下一个上山位置
                    while right < len(A) and A[right] <= A[right - 1]:
                        right += 1
                    left = right - 1
                else:
                    is_up = False
                    right += 1
            else:
                if A[right] > A[right - 1]:
                    result = max(result, right - left)
                    left = right - 1
                    right += 1
                    is_up = True
                elif A[right] == A[right - 1]:
                    result = max(result, right - left)
                    # 迭代寻找下一个上山位置
                    while right < len(A) and A[right] <= A[right - 1]:
                        right += 1
                    left = right - 1
                    is_up = True
                else:
                    right += 1

        # 迭代结束若为下山状态，更新最后的结果
        if not is_up:
            result = max(result, right - left)

        return result
```
